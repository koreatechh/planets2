>## **1.필요성**
>- 평상시 우리는 샤워나 세수를 할 때 대게 온수를 사용하며, 원하는 물의 온도로 바꾸기
위하여 아날로그식의 버튼을 움직여 온도를 조절한다. 그러는 와중에 조금 온수 
방향으로 틀었을 뿐인데도, 뜨거운 물이 쏟아져 나와 **화상을 입을 뻔 하는 상황에
처하는 경우**가 많으며 온도를 조절하는 그 2~5초 동안 **낭비되는 물의 양이 생각보다
많다**
